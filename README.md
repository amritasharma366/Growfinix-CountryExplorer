# Country Explorer

A simple Python application that allows users to explore country information and view the current weather in its capital city using live APIs.

## Features

- Search any country
- Display capital city
- Display region
- Display population
- Display currency
- Display current weather in the capital
- Display humidity
- Display wind speed

## Technologies Used

- Python
- Requests Library
- API Ninjas API
- Open-Meteo API

## How to Run

1. Install requirements

pip install -r requirements.txt

2. Run the project

python main.py

India
